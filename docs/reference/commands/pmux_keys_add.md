## pmux keys add

Add a client API key

```
pmux keys add [flags]
```

### Options

```
      --api-key-file string   read the key from a private file
      --api-key-stdin         read the key from standard input
      --generate              generate a random key and show it once
  -h, --help                  help for add
```
