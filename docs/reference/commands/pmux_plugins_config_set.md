## pmux plugins config set

Replace a plugin configuration with a JSON object

```
pmux plugins config set <plugin> <json> [flags]
```

### Options

```
  -h, --help    help for set
      --patch   shallow-merge instead of replace; null values delete keys
```

### Options inherited from parent commands

```
      --config-dir string   override the PMux config root
      --json                emit machine-readable JSON or NDJSON
      --verbose             write safe diagnostic details to stderr
      --yes                 accept the command's defined confirmation
```
