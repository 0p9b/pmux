## pmux setup

Set up or adopt CLIProxyAPI

```
pmux setup [flags]
```

### Options

```
      --config-path string   absolute CLIProxyAPI config path
      --harden               run the separately previewed adoption hardening transaction
  -h, --help                 help for setup
      --mode string          setup mode: managed or adopt
      --proxy-path string    absolute CLIProxyAPI executable path
```

### Options inherited from parent commands

```
      --config-dir string   override the PMux config root
      --json                emit machine-readable JSON or NDJSON
      --verbose             write safe diagnostic details to stderr
      --yes                 accept the command's defined confirmation
```
