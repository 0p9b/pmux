## pmux config edit

Edit one private temporary configuration copy

```
pmux config edit [flags]
```

### Options

```
      --editor string   editor executable
  -h, --help            help for edit
      --restart         restart after a restart-required change
```

### Options inherited from parent commands

      --yes                 accept the command's defined confirmation
