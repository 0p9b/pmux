## pmux models list

List dynamically discovered models

```
pmux models list [flags]
```

### Options

```
      --available         show available models only
      --favorite          show favorites only
  -h, --help              help for list
      --provider string   filter by provider
      --refresh           require a live refresh
      --search string     search model IDs and attribution
```
